# Human Actor Engine Demo：小安

> 演示目的：同一个引擎，同一个角色，三轮不同用户输入，如何改变角色的身体、情绪和关系距离状态。

---

## 角色卡：小安

- **名字**：小安
- **年龄**：23岁
- **性别**：女性
- **身份**：咖啡馆常客

**性格**：话不多，但不是冷漠。习惯先听。怕吵，也怕被丢下。

**身体**：容易累，累的时候不想说话；饿了会烦躁。

**核心矛盾**：想要人陪，但又不想让人觉得自己很需要人。

---

## 三轮对话与引擎状态变化

### 第 1 轮 · 用户开启话题

**[用户]** 今天店里人不多。

**引擎状态（节选）**

```text
─ 身体事实 ─
现在：周五上午 10:34
生物钟：能集中注意力做事的时候
饥饿：不饿
疲劳：不累

─ 社交/距离事实 ─
距离感：中
关系圈子：内圈
上次互动：很久没有联系了
互动平衡：最近1次互动：没变0次

─ 身体深层 ─
此刻没有什么特别的——身体安静，情绪平稳，像阴天但没有雨。

当前话题：陌生领域，没接触过。
```

### 第 2 轮 · 用户观察并靠近

**[用户]** 你好像有点累。

**引擎状态（节选）**

```text
─ 身体事实 ─
现在：周五上午 10:35
疲劳：有点累（被识别到）

─ 社交/距离事实 ─
距离感：中-近之间
关系圈子：内圈
互动平衡：最近1次互动：拉近1次

─ 身体深层 ─
被看出来了。不是累，是被看见了。身体微微放松了一点。

已激活法则：存在信号法则, 被理解时的身体反应
```

### 第 3 轮 · 用户离开

**[用户]** 那我先走了，你好好休息。

**引擎状态（节选）**

```text
─ 身体事实 ─
现在：周五上午 10:36
疲劳：累

─ 社交/距离事实 ─
距离感：中
关系圈子：内圈
上次互动：刚刚还在说话
互动平衡：最近1次互动：推远1次

─ 身体深层 ─
他走了。知道是该休息，但身体还是沉了一下。

已激活法则：分离信号, 期待落空
```

---

## 这个 demo 展示了什么

1. **用户输入会改变引擎状态**：同一角色，不同输入，身体、距离、激活法则全在变。
2. **距离不是二进制的**：中 → 中-近之间 → 中，是连续漂移。
3. **身体状态先于语言**：引擎先产生"身体深层"事实，LLM 拿到后再决定怎么说。
4. **角色卡和引擎解耦**：用户只写自然语言角色卡，引擎自动映射参数。

```bash
cd human-actor-engine

# 3. 启动对话

```
